## Requirements

* Ruby (3.1.0 or above)
* RSpec (3.0 or above)

## Installation

Install the gem by adding it to your `Gemfile`:

```bash
gem install govuk-rspec-helpers
```

## Setup

Add this to your `spec_helper.rb`:

```ruby
require "govuk_rspec_helpers"
```
